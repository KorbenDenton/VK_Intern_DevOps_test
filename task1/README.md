## Описание

Система мониторинга веб-приложения с автоматическим перезапуском при сбоях.  
Используется Nginx в качестве веб-сервера и bash-скрипт для мониторинга.

## Данный проект реализует систему мониторинга простого веб-приложения с функциями:

- Проверки доступности приложения по HTTP
- Логирования статусов и ошибок
- Автоматического перезапуска Nginx при недоступности приложения
- Таймер systemd для периодической проверки

## Компоненты

- **Nginx** — веб-сервер, отдающий страницу с "Hello World"
- **nginx.conf** - кастомный конфиг для nginx'а
- **monitoring.sh** - основной скрипт мониторинга
- **deploy.sh** — автоматическая установка и настройка
- **monitoring.service** - systemd service unit для запуска скрипта
- **monitoring.timer** - systemd timer unit для управления частотой запуска скрипта

## Проверка работы

Убедитесь, что таймер и веб-сервер активны:
```bash
systemctl status monitoring.timer
systemctl status nginx
```

Просмотр логов мониторинга:
```bash
tail -f /var/log/monitoring.log
```

Ручной запуск скрипта:
```bash
sudo /usr/local/bin/monitoring.sh
```
